## 프로젝트 실행 방법

git clone 레포지토리
클론된 레포지토리 폴더로 이동
yarn install
yarn start

node 또는 yarn이 설치되지 않은 경우 프로젝트 실행 전에 설치하여야 함
NodeJS 공식 홈페이지 : https://nodejs.org/ko/
Yarn Package Manager 공식 홈페이지 (Install Guide) : https://yarnpkg.com/getting-started/install
